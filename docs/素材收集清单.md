# 学校数字简介系统 - 素材收集清单

> 本清单由系统开发方提供，请学校相关部门按要求准备相应素材。

---

## 一、必填素材（开发前必须提供）

### 1.1 学校标识

| 素材名称 | 规格要求 | 用途说明 |
|---------|---------|---------|
| 学校Logo | PNG透明底，≥1MB，建议尺寸2000×2000px | 网站顶部、APP图标、二维码 |
| 学校简称 | 2-4个汉字，如"XX中学" | 首页展示、页面标题 |

### 1.2 学校基本信息

| 素材名称 | 格式要求 | 字数要求 | 用途说明 |
|---------|---------|---------|---------|
| 学校全称 | Word文档 | 20字以内 | 页面标题 |
| 学校简称 | Word文档 | 2-4字 | 首页展示 |
| 建校时间 | 文字 | - | 学校简介页 |
| 学校地址 | 文字，含省市区 | - | 联系方式页 |
| 联系电话 | 文字 | - | 联系方式页 |
| 招生电话 | 文字 | - | 首页展示 |
| 学校邮箱 | 文字 | - | 联系方式页 |
| 学校官网 | URL链接 | - | 联系方式页 |

### 1.3 学校简介内容

| 内容模块 | 字数要求 | 说明 |
|---------|---------|------|
| 学校概况/简介 | 500-1000字 | 整体介绍学校的定位、规模、特色 |
| 历史沿革 | 300-500字 | 学校发展历程、重要里程碑 |
| 办学理念 | 200-300字 | 办学宗旨、教育理念、培养目标 |
| 校训/校风/学风 | 简短文字 | 各1-2句 |
| 校园文化 | 300-500字 | 特色活动、传统节日、学生社团等 |
| 荣誉资质 | 列表形式 | 列出主要荣誉和称号 |

### 1.4 教师信息

| 信息字段 | 说明 | 示例 |
|---------|------|------|
| 姓名 | 必填 | 张三 |
| 性别 | 必填 | 男/女 |
| 职称 | 必填 | 高级教师/一级教师/二级教师 |
| 任教学科 | 必填 | 语文/数学/英语... |
| 学历 | 选填 | 本科/硕士/博士 |
| 教龄 | 选填 | 10年 |
| 个人简介 | 选填，200字以内 | 个人经历、教学特色等 |
| 荣誉奖项 | 选填 | 所获荣誉列表 |

**照片要求**：
- **证件照**：白色背景，JPG格式，≥500KB，建议尺寸800×1000px
- **生活照**：JPG格式，≥1MB，建议尺寸2000×1500px，每人2-3张

### 1.5 校园照片

| 类型 | 数量 | 规格要求 | 用途 |
|------|------|---------|------|
| 校门照片 | 2-3张 | JPG，≥3MB，建议3000×2000px | 首页展示、全景拍摄参考 |
| 教学楼全景 | 2-3张/栋 | JPG，≥2MB | 相册展示 |
| 操场/运动场 | 3-5张 | JPG，≥2MB | 全景拍摄、风采展示 |
| 实验室照片 | 3-5张/间 | JPG，≥2MB | 特色部室展示 |
| 图书馆/阅览室 | 3-5张 | JPG，≥2MB | 全景拍摄、风采展示 |
| 食堂照片 | 3-5张 | JPG，≥2MB | 风采展示 |
| 校园风景 | 5-10张 | JPG，≥2MB | 校园相册 |
| 其他特色场所 | 3-5张 | JPG，≥2MB | 特色展示 |

## 三、全景拍摄清单（74亩学校建议）

### 3.1 必拍点位

| 序号 | 拍摄位置 | 拍摄数量 | 备注 |
|------|---------|---------|------|
| 1 | 校门口 | 1 | 入口处全景 |
| 2 | 教学楼A栋 | 2 | 每层1个点位 |
| 3 | 教学楼B栋 | 2 | 每层1个点位 |
| 4 | 实验楼 | 3 | 化学/物理/生物各1 |
| 5 | 操场/跑道 | 2 | 全景+看台 |
| 6 | 体育馆/风雨操场 | 1 | 室内全景 |
| 7 | 图书馆/阅览室 | 1-2 | 室内全景 |
| 8 | 食堂 | 1 | 室内全景 |
| 9 | 行政楼/会议室 | 1 | 室内全景 |
| 10 | 校园景观区 | 2 | 花园/绿化/雕塑 |

**建议总点位：14-18个**

### 3.2 拍摄时间建议

| 季节 | 最佳时段 | 说明 |
|------|---------|------|
| 春季(3-5月) | 上午9-11点，下午3-5点 | 光线柔和，适合拍建筑 |
| 秋季(9-11月) | 上午9-11点，下午3-5点 | 天高气爽，色彩丰富 |
| 夏季 | 上午7-9点，下午5-7点 | 避开正午强光 |
| 冬季 | 上午10-12点，下午2-4点 | 光照时间短 |

**建议拍摄时长**：半天至1天（视点位数量）

## 五、注意事项

### 5.1 照片质量要求

- ✅ 清晰度高，无模糊
- ✅ 光线充足，无严重阴影
- ✅ 构图合理，突出主体
- ✅ 背景整洁，无杂物
- ❌ 不要使用美颜/滤镜过度的照片
- ❌ 不要使用有水印的照片

### 5.2 版权说明

- 所有素材须为学校自有版权或已获得授权
- 涉及学生、教职工的照片需获得本人同意
- 荣誉证书需为学校正式获得

### 5.3 提交时间

请在项目启动后 **2周内** 完成所有素材准备并提交。
